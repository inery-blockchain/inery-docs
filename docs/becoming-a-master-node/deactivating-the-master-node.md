# Deactivating the Master Node

<br/>

To deactivate your master node capabilities execute:

<br/>

```shell
cline system makeprod unapprove YOUR_ACCOUNT_NAME YOUR_ACCOUNT_NAME
```

<br/>

This removes you from the master node schedule list.
